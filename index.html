HTML y CSS

publicado 01-03-2026

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar utilizado para estructurar y organizar el contenido de las páginas web mediante etiquetas o marcas. No es un lenguaje de programación, sino un lenguaje de marcado que define elementos como títulos, párrafos, imágenes y enlaces. CSS son las siglas en inglés de Cascading Style Sheets, traducido al español como Hojas de Estilo en Cascada. Es un lenguaje declarativo fundamental en el desarrollo web utilizado para definir el aspecto visual, diseño, formato y estilo de documentos estructurados en HTML o XML, separando el contenido de la presentación.
